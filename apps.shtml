There are some applications implemented with the help of python-ldap (in alphabetical order):
Kantan

A web-based searching and browsing tool.

LDAP Configuration Tool

LDAP Configuration Tool is a simple application that helps you reconfigure your Linux box to take advantage of an LDAP directory for access control, account management, and user authentication.

LDAPUserFolder

The LDAP User Folder is a user folder replacement for Zope that authenticates Zope users against LDAP. It is written for Zope versions 2.4.0 and higher.

proxy_auth_ldap

A simple script for authenticating users accessing a Squid proxy.

pyCA

A bunch of Python scripts for setting up a X.509 CA. Uploading certificates to LDAP servers is done via python-ldap.

web2ldap

web2ldap is a full-featured web-based LDAPv3 client with schema support/schema browser.
