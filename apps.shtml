There are some applications implemented with the help of python-ldap (in alphabetical order):
CMFLDAP

CMFLDAP is a Zope CMF product that replaces the standard membership management tools (Registration Tool, Membership Tool, Member Data Tool) with a set that speaks LDAP. It requires a working LDAPLoginAdapter and a working LDAPUserManager.

Kantan

A web-based searching and browsing tool.

LDAPLoginAdapter and LDAPUserManager

The LDAPLoginAdapter is a user folder replacement for Zope that authenticates Zope users against LDAP. It is written for Zope versions 2.3.0 and higher.

The LDAPUserManager is a LDAP user record administration tool to be used in conjunction with the LDAPLoginAdapter. It is written for Zope versions 2.3.0 and higher.

proxy_auth_ldap

A simple script for authenticating users accessing a Squid proxy.

pyCA

A bunch of Python scripts for setting up a X.509 CA. Uploading certificates to LDAP servers is done via python-ldap.

web2ldap

A web gateway to LDAP servers.
