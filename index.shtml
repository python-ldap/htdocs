What is Python-LDAP?

Python-LDAP provides a module for accessing LDAP servers from Python. It wraps an underlying LDAP C library that provides an RFC 1823 API, such as OpenLDAP.

Mailing list

Discussion about the use and future of Python-LDAP occurs in the python-ldap-dev mailing list. You can subscribe or unsubscribe to this list or browse the list archive.
