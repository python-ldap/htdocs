Because distributions seem to be all over the place, this page tries to list all the current ones we know of.

Official releases

The project release area has the latest 'official' releases.

Source

  • Anonymous CVS instructions (go here to be really up-to-date!)
  • Daily snapshot of CVS archive (mainly for backup purposes)
  • 1.10alpha3 released source (outdated, does not build with OpenLDAP 2 libs!)

Binaries

Red Hat Linux

  • RPMs for Red Hat 7.2 maintained by Joe Little

S.u.S.E. Linux

S.u.S.E. Linux ships with python-ldap. They use a rather old version in 7.x though. Version 8.0 comes with recent python-ldap.

Debian Linux

  • Debian package python-ldap 1.8-1
    (based on outdated python-ldap 1.8, needs OpenLDAP 1 libs)
  • Debian package 1.9.999.pre04-1
    (based on python-ldap pre-release 2.0.0pre4, needs OpenLDAP 2.0.21+ libs)

Win32

You can find some pre-compiled packages for Windows 32 platform (1.10alpha3 based on UMich LDAP libs).
