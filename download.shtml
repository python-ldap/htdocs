Binaries

Red Hat Linux

  • RPMs for Red Hat Linux maintained by Joe Little

S.u.S.E. Linux

S.u.S.E. Linux ships with python-ldap. They use a rather old version in 7.x though. Version 8.0 ships with python-ldap 2.0.0pre04.

Debian Linux

  • Debian package python-ldap 1.8-1
    (based on outdated python-ldap 1.8, needs OpenLDAP 1 libs)
  • Debian package 1.9.999.pre04-1
    (based on python-ldap pre-release 2.0.0pre4, needs OpenLDAP 2.0.21+ libs)

Win32

You can find some pre-compiled packages for Windows 32 platform (1.10alpha3 based on UMich LDAP libs).
