2. Can I do a LDAPv3 bind via python-ldap?
In general: No.
In detail: Depends on the underlying LDAP lib.
3. Can python-ldap can be built against OpenLDAP 2.0.x libs?
No, since OpenLDAP 2.0.x implements a slightly different API (read more) python-ldap cannot be linked against this version. There are two patches available which work with CVS snapshot as of May 2001. See also download page.
